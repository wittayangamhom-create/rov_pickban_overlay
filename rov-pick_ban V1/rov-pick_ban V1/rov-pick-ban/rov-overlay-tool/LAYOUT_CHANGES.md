# 🎨 Layout Changes - เวอร์ชันใหม่

## การเปลี่ยนแปลงหลัก

### 1. ย่อ Pick Draft ให้พอดีกรอบหน้าจอ ✅

**เปลี่ยนแปลง:**
- ลดขนาดช่องฮีโร่จาก 240x390 → **220x360 pixels**
- ลดระยะห่างระหว่างช่องจาก 15px → **12px**
- เพิ่ม padding ด้านข้าง 50px และด้านล่าง 20px
- ลบพื้นหลัง gradient สีดำ

**ผลลัพธ์:**
- Pick slots พอดีกับหน้าจอ 1920x1080
- ดูกระชับและเป็นระเบียบมากขึ้น
- มีพื้นที่ว่างด้านข้างพอดี

---

### 2. ย้ายชื่อทีมและโลโก้มาตรงกลาง ✅

**เปลี่ยนแปลง:**
- สร้างส่วนใหม่ตรงกลางระหว่าง Blue และ Red picks
- แสดงโลโก้ทีม (80x80px)
- แสดงชื่อทีมด้านล่างโลโก้
- มี border สีตามทีม (น้ำเงิน/แดง)
- มีเงาและเอฟเฟกต์ blur backdrop

**ตำแหน่ง:**
- อยู่ตรงกลางหน้าจอ
- สูงจากด้านล่าง 80px
- มีช่องว่างระหว่างทีม 40px

**การออกแบบ:**
```
┌─────────────────────────────────────────────┐
│                                             │
│    [Blue Picks 5 ช่อง]                      │
│                                             │
│           [Blue Card]  [Red Card]           │
│           โลโก้ + ชื่อ   โลโก้ + ชื่อ         │
│                                             │
│                      [Red Picks 5 ช่อง]     │
└─────────────────────────────────────────────┘
```

## 📐 ขนาดใหม่

### Pick Slots
- **ขนาด:** 220 x 360 pixels (ลดลงจากเดิม)
- **ระยะห่าง:** 12px
- **จำนวน:** 5 slots ต่อทีม

### Team Center Cards
- **โลโก้:** 80 x 80 pixels
- **ชื่อทีม:** Font-size 28px
- **Padding:** 15px 25px
- **Border:** 3px solid

### Layout Spacing
- **ด้านข้าง:** 50px
- **ด้านล่าง:** 20px
- **Center cards:** 80px จากด้านล่าง

## 🎯 จุดเด่นของ Layout ใหม่

### 1. กระชับขึ้น
- ไม่มีส่วนล้นออกนอกจอ
- ใช้พื้นที่อย่างมีประสิทธิภาพ

### 2. มองเห็นชัดเจน
- ชื่อทีมและโลโก้อยู่ตรงกลาง
- แยกทีมชัดเจนด้วยสี

### 3. ดูมืออาชีพ
- Layout สมดุล
- มี visual hierarchy ชัดเจน
- เอฟเฟกต์สวยงาม

## 💡 การปรับแต่งเพิ่มเติม

### ปรับขนาดช่องฮีโร่

```css
/* ถ้าต้องการให้เล็กกว่านี้ */
.pick-slot {
    width: 200px;
    height: 340px;
}
```

### ปรับตำแหน่งชื่อทีม

```css
/* ย้ายขึ้นหรือลง */
.team-center-info {
    bottom: 100px;  /* เปลี่ยนจาก 80px */
}
```

### ปรับขนาดโลโก้

```css
.team-center-logo {
    width: 100px;   /* เปลี่ยนจาก 80px */
    height: 100px;
}
```

### ซ่อนชื่อทีมตรงกลาง

```css
.team-center-name {
    display: none;
}
```

### เปลี่ยนสี Border

```css
.team-center-card.blue {
    border-color: #00ffff;  /* สีฟ้าสดกว่า */
}

.team-center-card.red {
    border-color: #ff0066;  /* สีแดงสดกว่า */
}
```

## 🐛 Troubleshooting

**ปัญหา: Picks ยังล้นกรอบ**
- ลดขนาด `.pick-slot` ลงอีก
- ลด gap ระหว่างช่อง
- เพิ่ม padding ด้านข้าง

**ปัญหา: ชื่อทีมไม่แสดง**
- ตรวจสอบ JavaScript ทำงานถูกต้อง
- Refresh หน้าด้วย Ctrl+F5
- เช็ค Console ใน DevTools

**ปัญหา: โลโก้ไม่แสดง**
- ตรวจสอบชื่อไฟล์ใน team-logos/
- ตรวจสอบ path ในโค้ด
- ดู error ใน Console

## 📏 ข้อกำหนดรูป

- **ขนาดแนะนำ:** 240 x 390 pixels
- **รูปแบบไฟล์:** PNG (รองรับความโปร่งใส)
- **ชื่อไฟล์:** ตัวเล็กทั้งหมด (lowercase)
- **ขนาดไฟล์:** ไม่ควรเกิน 200KB ต่อรูป
- **คุณภาพ:** สูง, ไม่เบลอ
- **พื้นหลัง:** โปร่งใสหรือสีพื้น (แนะนำโปร่งใส)

## 💡 วิธีตั้งชื่อไฟล์

**สำคัญ: ชื่อไฟล์ต้องเป็นตัวเล็กทั้งหมด**

ตัวอย่าง:
- ถูก: airi.png, nakroth.png, tel'annas.png
- ผิด: Airi.png, Nakroth.png, Tel'Annas.png

### Windows:
1. คลิกขวาที่ไฟล์ → Rename
2. Copy ชื่อจากรายการนี้วาง
3. ตรวจสอบว่าเป็นตัวเล็กทั้งหมด

### Mac:
1. คลิกที่ไฟล์ → Enter
2. Copy ชื่อจากรายการนี้วาง
3. ตรวจสอบว่าเป็นตัวเล็กทั้งหมด

## ✅ วิธีตรวจสอบ

รันคำสั่ง:
```bash
npm run check-heroes
```

จะแสดงรายการว่ารูปใดมีแล้ว รูปใดยังไม่มี

## 📁 โฟลเดอร์ที่วางรูป

```
rov-overlay-tool/public/images/heroes/
```

วางรูปทั้งหมด 125 รูป (หรือเท่าที่จะใช้) ในโฟลเดอร์นี้

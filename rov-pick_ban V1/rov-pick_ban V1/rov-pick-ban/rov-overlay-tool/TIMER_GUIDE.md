# ⏱️ คู่มือการใช้งาน Timer

## ฟีเจอร์ Timer ที่อัปเดตใหม่

Timer ตอนนี้มีความสามารถเต็มรูปแบบ พร้อมระบบนับถอยหลังอัตโนมัติ!

---

## 🎯 ฟีเจอร์หลัก

- ✅ **นับถอยหลังอัตโนมัติ** - เริ่มได้ด้วยปุ่มเดียว
- ✅ **Pause/Resume** - หยุดและเล่นต่อได้
- ✅ **Reset** - รีเซ็ตกลับค่าเริ่มต้น
- ✅ **Presets** - เลือกเวลาสำเร็จรูป (30s, 1m, 2m, 3m, 5m)
- ✅ **สีเปลี่ยนตามเวลา** - เตือนเมื่อเวลาใกล้หมด
- ✅ **แสดงผล Real-time** - อัปเดตไปที่ Overlay ทันที
- ✅ **Alert เมื่อหมดเวลา** - แจ้งเตือนเมื่อ Timer เหลือ 0

---

## 🎨 ส่วนประกอบ

### 1. Timer Display (หน้าจอแสดงผล)
```
┌─────────────┐
│   00:17     │  ← หน้าจอแสดงเวลาขนาดใหญ่
└─────────────┘
```
- แสดงเวลาปัจจุบัน
- เปลี่ยนสีตามสถานะ:
  - **เขียว** - กำลังนับ (Running)
  - **ส้ม** - เหลือ 10 วินาที (Warning)
  - **แดง** - เหลือ 5 วินาที (Danger)
  - **เทา** - หยุดนับ (Paused/Stopped)

### 2. Timer Input
- ใส่เวลาในรูปแบบ `MM:SS`
- ตัวอย่าง: `01:30`, `00:45`, `02:00`

### 3. ปุ่มควบคุม
- **Set Timer** - ตั้งเวลาที่ต้องการ
- **▶️ Start** - เริ่มนับถอยหลัง
- **⏸️ Pause** - หยุดชั่วคราว
- **⏹️ Reset** - รีเซ็ตกลับค่าเริ่มต้น

### 4. Presets (เวลาสำเร็จรูป)
- **30s** - 30 วินาที
- **1m** - 1 นาที
- **2m** - 2 นาที
- **3m** - 3 นาที
- **5m** - 5 นาที

## 📝 วิธีใช้งาน

### 🎯 การใช้งานพื้นฐาน

#### 1. ตั้งเวลาด้วยตัวเอง
```
1. พิมพ์เวลาใน Input: 01:30
2. คลิก "Set Timer"
3. คลิก "▶️ Start"
4. Timer จะเริ่มนับถอยหลัง
```

#### 2. ใช้ Preset (แนะนำ)
```
1. คลิกปุ่ม "1m" (หรือเวลาอื่นที่ต้องการ)
2. คลิก "▶️ Start"
3. Timer จะเริ่มนับถอยหลังทันที
```

#### 3. หยุดชั่วคราว
```
1. คลิก "⏸️ Pause"
2. Timer จะหยุดนับ
3. คลิก "▶️ Start" อีกครั้งเพื่อเล่นต่อ
```

#### 4. รีเซ็ต
```
1. คลิก "⏹️ Reset"
2. Timer จะกลับไปค่าเริ่มต้น
3. พร้อมเริ่มใหม่
```

## 🎮 Use Cases

### Scenario 1: Draft Pick (1 นาที)
```
1. คลิก "1m"
2. คลิก "Start"
3. ทีมมีเวลา 1 นาทีในการเลือกฮีโร่
4. เมื่อเวลาใกล้หมด ตัวเลขจะเปลี่ยนเป็นสีส้มและแดง
5. หมดเวลา → Alert แจ้งเตือน
```

### Scenario 2: Ban Phase (30 วินาที)
```
1. คลิก "30s"
2. คลิก "Start"
3. ทีมมีเวลา 30 วินาทีในการ Ban
4. หมดเวลา → ยืนยัน Reset → เริ่มใหม่
```

### Scenario 3: Pause ระหว่างทาง
```
1. กำลังนับอยู่ 00:45
2. มีปัญหาเกิดขึ้น
3. คลิก "Pause"
4. แก้ไขปัญหา
5. คลิก "Start" เล่นต่อ
```

### Scenario 4: เปลี่ยนเวลากลางคัน
```
1. กำลังนับอยู่ 00:45
2. ต้องการเปลี่ยนเป็น 02:00
3. คลิก "2m"
4. คลิก "Start"
5. Timer จะเริ่มที่ 02:00
```

## 🎨 สีและสัญญาณเตือน

| เวลาเหลือ | สี | สถานะ |
|-----------|-----|-------|
| > 10 วินาที | เขียว 🟢 | ปกติ |
| 6-10 วินาที | ส้ม 🟠 | Warning (กระพริบ) |
| 1-5 วินาที | แดง 🔴 | Danger (กระพริบเร็ว) |
| 0 วินาที | - | Alert! |

## ⚙️ การตั้งค่าขั้นสูง

### เปลี่ยนเวลา Preset

แก้ไขใน `control.html`:
```html
<button class="btn btn-small" onclick="setTimerValue('00:15')">15s</button>
<button class="btn btn-small" onclick="setTimerValue('10:00')">10m</button>
```

### เปลี่ยนเกณฑ์เตือน

แก้ไขใน `control.js`:
```javascript
// เปลี่ยนจาก 10 วินาที เป็น 15 วินาที
```

## 🔧 Troubleshooting

### ปัญหา: Timer ไม่นับ
**วิธีแก้:**
1. ตรวจสอบว่ากด "Start" แล้ว
2. ลอง Refresh หน้า (F5)
3. ตรวจสอบว่าไม่มี Error ใน Console (F12)

### ปัญหา: Overlay ไม่อัปเดต
**วิธีแก้:**
1. ตรวจสอบว่า Server รันอยู่
2. Refresh หน้า Overlay (Ctrl+F5)
3. ตรวจสอบ Socket Connection

### ปัญหา: Timer กระตุก
**วิธีแก้:**
1. ปิดแท็บอื่นที่ไม่ใช้
2. ตรวจสอบ CPU Usage
3. Restart Browser

### ปัญหา: Timer ไม่ Reset
**วิธีแก้:**
1. คลิก "Reset" อีกครั้ง
2. Refresh หน้า
3. ตั้งเวลาใหม่ด้วย Preset

## 💡 Tips & Tricks

### 1. ใช้ Keyboard Shortcuts
```
Tab → เปลี่ยนไปยัง Input
Enter → Set Timer (เมื่ออยู่ใน Input)
Space → Start/Pause (ถ้าตั้งค่า)
```

### 2. ตั้งเวลาที่ชอบไว้
สร้าง Preset เพิ่มสำหรับเวลาที่ใช้บ่อย

### 3. ดู Timer บน Overlay
- Overlay จะแสดง Timer แบบ Real-time
- ผู้ชมจะเห็นเวลาที่เหลือ

### 4. ใช้ร่วมกับ Draft
- ตั้งเวลาก่อนเริ่ม Draft
- Start พร้อมกับทีมเริ่มเลือก
- Pause ถ้ามีปัญหาเกิดขึ้น

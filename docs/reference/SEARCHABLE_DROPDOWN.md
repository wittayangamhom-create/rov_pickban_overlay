# 🔍 คู่มือ Searchable Dropdown

## ฟีเจอร์ใหม่: พิมพ์ชื่อค้นหาใน Dropdown

ตอนนี้ Dropdown ทุกช่องสามารถ**พิมพ์ชื่อเพื่อค้นหา**ฮีโร่ได้แล้ว!

---

## 🎯 วิธีใช้งาน

### แบบพิมพ์ค้นหา (แนะนำ)

1. **คลิกที่ Dropdown** → เปิดรายการ
2. **พิมพ์ชื่อฮีโร่** → เช่น `nak`, `violet`, `tel`
3. **รายการกรอง** → แสดงเฉพาะฮีโร่ที่ตรง
4. **เลือกฮีโร่** → คลิกหรือกด Enter
5. **เสร็จ!** → ฮีโร่ถูกเลือก

### แบบเลื่อนหา (เหมือนเดิม)

1. **คลิกที่ Dropdown**
2. **เลื่อนหา** → scroll ลงไป
3. **เลือก** → คลิกฮีโร่ที่ต้องการ

---

## ⌨️ Keyboard Shortcuts

| ปุ่ม | ทำอะไร |
|------|--------|
| **พิมพ์** | กรองฮีโร่ทันที |
| **↓** | เลื่อนลงรายการ |
| **↑** | เลื่อนขึ้นรายการ |
| **Enter** | เลือกฮีโร่ที่ highlight |
| **Esc** | ปิด Dropdown |
| **Backspace** | ลบตัวอักษร |

## 🎨 ตัวอย่างการใช้งาน

### Scenario 1: เลือก Nakroth แบบรวดเร็ว
```
1. คลิก Dropdown "Blue Team - Picks (Slot 1)"
2. พิมพ์: "nak"
3. เห็น: nakroth
4. กด: Enter
5. เสร็จ!
```

### Scenario 2: Ban Violet
```
1. คลิก Dropdown "Blue Team - Bans (Ban 1)"
2. พิมพ์: "vio"
3. เห็น: violet, volkath (2 ตัว)
4. กด: ↓ เลือก violet
5. กด: Enter
```

### Scenario 3: ค้นหาฮีโร่ที่มี apostrophe
```
1. คลิก Dropdown
2. พิมพ์: "tel'a" หรือ "tela"
3. เห็น: tel'annas
4. คลิก
```

## 💡 Tips & Tricks

### 1. พิมพ์บางส่วนก็พอ
- `nak` → nakroth
- `vio` → violet
- `tel` → tel'annas
- `bolt` → bolt baron

### 2. ไม่ต้องกังวล Case
- `NAKROTH` = `nakroth` = `Nakroth`
- ระบบไม่สนตัวเล็ก-ใหญ่

### 3. ใช้ลูกศรเลือกเร็ว
```
1. พิมพ์ชื่อ
2. กด ↓ (ถ้ามีหลายตัว)
3. กด Enter
```

### 4. ล้างการเลือก
- คลิก **X** ข้าง Dropdown
- หรือเลือก "Select Hero" (ตัวแรก)

### 5. ค้นหาตัวอักษรพิเศษ
- `tel'` → tel'annas
- `azzen'` → azzen'ka
- `d'ar` → d'arcy
- `bolt` → bolt baron (มีช่องว่าง)

## 🎯 ข้อดี

### 1. เร็วกว่าเดิมมาก
- **Dropdown ธรรมดา:** เลื่อนหา 125 ตัว (~10 วินาที)
- **Searchable:** พิมพ์ 3 ตัว (~2 วินาที)
- **ประหยัดเวลา 80%!**

### 2. ใช้งานง่าย
- พิมพ์ไม่กี่ตัวก็เจอ
- ไม่ต้องเลื่อนหานาน
- รองรับ Keyboard

### 3. ดูสวยงาม
- UI สวยกว่า Dropdown ธรรมดา
- มี Highlight effect
- Responsive

### 4. ทำงานในทุก Slot
- ✅ Blue Team - Picks (5 Slots)
- ✅ Blue Team - Bans (4 Slots)
- ✅ Red Team - Picks (5 Slots)
- ✅ Red Team - Bans (4 Slots)

## 🐛 Troubleshooting

### ปัญหา: พิมพ์แล้วไม่กรอง
**วิธีแก้:**
1. ตรวจสอบว่า Select2 โหลดแล้ว
2. Refresh หน้า (F5)
3. Clear Cache (Ctrl+F5)

### ปัญหา: Dropdown ไม่เปิด
**วิธีแก้:**
1. คลิกอีกครั้ง
2. Refresh หน้า
3. ตรวจสอบ Console (F12)

### ปัญหา: รายการไม่ครบ
**วิธีแก้:**
1. ตรวจสอบว่าโหลดฮีโร่ครบ
2. ดูใน Console หา Error
3. Restart Server

### ปัญหา: พิมพ์ภาษาไทยไม่ได้
**คำตอบ:**
- Dropdown ค้นหาชื่อภาษาอังกฤษเท่านั้น
- พิมพ์ชื่อภาษาอังกฤษแทน

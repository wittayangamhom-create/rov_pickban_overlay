# 🔄 คู่มือการสลับชื่อผู้เล่น (Swap Players)

## ฟีเจอร์ใหม่: สลับชื่อ Player รวดเร็ว

ตอนนี้สามารถสลับชื่อผู้เล่นระหว่าง Position ต่างๆ ได้ด้วยปุ่มเดียว!

---

## 🎯 ปุ่มที่มีให้ใช้

### แต่ละทีมมี 6 ปุ่ม:

| ปุ่ม | ทำอะไร |
|------|--------|
| **1 ↔️ 2** | สลับชื่อ Player 1 กับ Player 2 |
| **2 ↔️ 3** | สลับชื่อ Player 2 กับ Player 3 |
| **3 ↔️ 4** | สลับชื่อ Player 3 กับ Player 4 |
| **4 ↔️ 5** | สลับชื่อ Player 4 กับ Player 5 |
| **1 ↔️ 3** | สลับชื่อ Player 1 กับ Player 3 |
| **1 ↔️ 5** | สลับชื่อ Player 1 กับ Player 5 |

---

## 📝 วิธีใช้งาน

### ขั้นตอนพื้นฐาน:

1. **หาส่วน Players** ในทีมที่ต้องการ (Blue/Red)
2. **คลิกปุ่มสลับ** เช่น "1 ↔️ 2"
3. **ชื่อสลับทันที** และอัปเดตไปที่ Overlay
4. **เสร็จ!** ไม่ต้องกด Update Players

---

## 🎮 ตัวอย่างการใช้งาน

### Scenario 1: สลับ Jungler กับ Mid
```
ก่อน:
Player 1: TOP (Nakroth)
Player 2: JUNGLE (Zhan)
Player 3: MID (Zhenzhe)

คลิก: "2 ↔️ 3"

หลัง:
Player 1: TOP (Nakroth)
Player 2: MID (Zhenzhe)    ← สลับแล้ว
Player 3: JUNGLE (Zhan)    ← สลับแล้ว
```

### Scenario 2: สลับ Top กับ ADC
```
ก่อน:
Player 1: TOP (Amily)
Player 2: JUNGLE (Murad)
Player 3: MID (Tulen)
Player 4: ADC (Violet)
Player 5: SUPPORT (Ormarr)

คลิก: "1 ↔️ 4"  
(ต้องคลิก 2 ครั้ง: 1↔️2, 2↔️3, 3↔️4)
หรือใช้วิธีอื่น...

หรือง่ายกว่า:
1. คลิก "1 ↔️ 2" → TOP กับ JUNGLE สลับ
2. คลิก "2 ↔️ 3" → เดิม TOP (ตอนนี้อยู่ P2) สลับกับ MID
3. คลิก "3 ↔️ 4" → เดิม TOP (ตอนนี้อยู่ P3) สลับกับ ADC

(หมายเหตุ: สำหรับการสลับที่ห่างกัน แนะนำให้สลับแบบเรียงลำดับ)
```

### Scenario 3: สลับ Top กับ Support (1 ↔️ 5)
```
ก่อน:
Player 1: TOP (Ryoma)
Player 5: SUPPORT (Thane)

คลิก: "1 ↔️ 5"

หลัง:
Player 1: SUPPORT (Thane)  ← สลับแล้ว
Player 5: TOP (Ryoma)      ← สลับแล้ว
```

## 💡 Use Cases

### 1. เปลี่ยน Role กลางเกม
```
ทีมต้องการสลับ Role:
- Top ไป Jungle
- Jungle ไป Top

คลิก: "1 ↔️ 2"
```

### 2. ผู้เล่นเปลี่ยนตำแหน่ง
```
Player เปลี่ยน Position:
- Position 2 → Position 4
- Position 4 → Position 2

คลิก: "2 ↔️ 3"
คลิก: "3 ↔️ 4"
```

### 3. แก้ไขความผิดพลาด
```
ใส่ชื่อผิด Position:
- TOP อยู่ที่ P3 (ควรอยู่ P1)
- MID อยู่ที่ P1 (ควรอยู่ P3)

คลิก: "1 ↔️ 3"
```

## 🎨 Visual Feedback

### การแสดงผล:

1. **ก่อนคลิก:**
   - ช่อง Input ปกติ (พื้นขาว)

2. **เมื่อคลิกปุ่มสลับ:**
   - ช่อง Input ทั้ง 2 กะพริบสีเหลือง
   - แสดงเวลา 0.5 วินาที

3. **หลังสลับ:**
   - กลับเป็นสีปกติ
   - ข้อความแสดง: "Swapped Player X ↔️ Player Y"

## ⚙️ ฟีเจอร์พิเศษ

### 1. Auto-Update
- สลับแล้วอัปเดตอัตโนมัติ
- ไม่ต้องกดปุ่ม "Update Players"
- ส่งไปยัง Overlay ทันที

### 2. Animation
- ช่อง Input กะพริบสีเหลือง
- บอกว่าสลับแล้ว
- ดูชัดเจน

### 3. Status Message
- แสดงข้อความยืนยัน
- บอกว่าสลับ Player ไหนบ้าง
- อยู่ที่มุมขวาบน

## 🔧 Tips & Tricks

### 1. การสลับหลายครั้ง
```
ต้องการย้าย Player 1 → Position 4:
คลิก: "1 ↔️ 2"
คลิก: "2 ↔️ 3"  
คลิก: "3 ↔️ 4"
```

### 2. การสลับแบบวงกลม
```
P1 → P2 → P3 → P1:
คลิก: "1 ↔️ 2"
คลิก: "2 ↔️ 3"
คลิก: "1 ↔️ 3"
```

### 3. การตรวจสอบ
- ดูที่ Overlay เพื่อยืนยัน
- ชื่อผู้เล่นเปลี่ยนทันที
- Position number ยังคงเดิม

## 📊 เปรียบเทียบวิธีเดิม vs ใหม่

| การทำงาน | วิธีเดิม | วิธีใหม่ |
|----------|---------|---------|
| สลับชื่อ | Copy-Paste ด้วยมือ | คลิกปุ่มเดียว |
| เวลาที่ใช้ | ~20 วินาที | ~2 วินาที |
| ขั้นตอน | 6-8 ขั้นตอน | 1 ขั้นตอน |
| อัปเดต | กดปุ่ม Update | อัตโนมัติ |
| ความเสี่ยงผิด | สูง | ต่ำ |

**ประหยัดเวลา 90%!**

## 🐛 Troubleshooting

### ปัญหา: สลับแล้วไม่เห็นผล
**วิธีแก้:**
1. Refresh หน้า Overlay (Ctrl+F5)
2. ตรวจสอบ Socket Connection
3. ดู Console หา Error (F12)

### ปัญหา: ปุ่มไม่ทำงาน
**วิธีแก้:**
1. Refresh หน้า Control Panel (F5)
2. ตรวจสอบว่า Server รันอยู่
3. ลอง Restart Server

### ปัญหา: ชื่อหายหลังสลับ
**วิธีแก้:**
1. ไม่ควรเกิดขึ้น ถ้าเกิด = Bug
2. Reload State จาก Server
3. ใส่ชื่อใหม่ด้วยมือ

## 💡 Best Practices

### 1. ใช้ปุ่มที่เหมาะสม
```
สลับ Position ติดกัน → ใช้ "X ↔️ X+1"
สลับ Position ห่าง → ใช้ "1 ↔️ 5" หรือ "1 ↔️ 3"
```

### 2. ตรวจสอบก่อนแข่ง
```
1. สลับชื่อให้ถูกต้อง
2. ดูที่ Overlay เพื่อยืนยัน
3. แก้ไขถ้าผิด
```

### 3. ใช้ร่วมกับ Switch Teams
```
1. สลับชื่อภายในทีม (Swap Players)
2. สลับทั้งทีม (Switch Teams)
3. ทำให้ได้ทุกแบบ
```
